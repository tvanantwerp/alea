Alea Password Generator

Alea is a simple password generator based on selecting several words at random which are easy to memorize but hard to guess.

Alea uses this wordlist hosted by Arnold G. Reinhold.

Alea uses JavaScript to select words from a large list to create your random password. This generator is a proof of concept. A skilled adversary could compromise this generator, so you should not totally trust your results.
